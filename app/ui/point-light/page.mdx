# Point Light
The `PointLight` component creates a rotating light. Dependencies needed:

```bash
npm install motion/react
```

## Source Code

```tsx
"use client";

import React from "react";
import { motion } from "motion/react";
import { cn } from "@/utils";

interface PointLightProps {
  angle?: number;
  intensity?: number;
  color?: string;
  className?: string;
}

export const PointLight: React.FC<PointLightProps> = ({
  angle = -30,
  intensity = 1,
  color = "white",
  className,
}) => {
  return (
    <motion.div
      className={cn(`absolute left-0 top-0 h-[150%] blur-xl origin-top-left ease-in-out duration-[2000ms]`, className)}
      style={{
        backgroundImage: `linear-gradient(${color}, transparent)`,
        width: `${intensity * 8}px`,
      }}
      initial={{ rotate: -angle }}
      animate={{ rotate: angle }}
    />
  );
};

```


## Props

| **Prop Name**     | **Type**                      | **Required** | **Default**   | **Description**                                                                 |
|-------------------|-------------------------------|--------------|---------------|---------------------------------------------------------------------------------|
| `angle`           | `number`                      | No           | `-30`         | The initial rotation angle of the point light in degrees.                       |
| `intensity`       | `number`                      | No           | `1`           | The intensity of the light, controlling the size of the light's gradient effect. |
| `color`           | `string`                      | No           | `"white"`     | The color of the light gradient. Can be any valid CSS color value.              |
| `className`       | `string`                      | No           | `undefined`   | Additional custom CSS classes to apply to the component.                        |
