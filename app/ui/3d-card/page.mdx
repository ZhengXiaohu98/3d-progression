# 3D Rotate

The `3D Rotate` component adds a 3D rotation effect to its child elements when hovered. The rotation dynamically responds to the user's mouse position.


## Example

<ExampleBox>
  <ThreeDRotate>
    <div className="w-40 h-40 border">
    </div>
  </ThreeDRotate>
</ExampleBox>


## Props

| Prop Name | Type   | Default Value | Description                                                                 |
|-----------|--------|---------------|-----------------------------------------------------------------------------|
| `xAngle`  | number | `15`          | Maximum rotation angle along the X-axis (vertical tilt).                    |
| `yAngle`  | number | `15`          | Maximum rotation angle along the Y-axis (horizontal tilt).                  |
| `zAngle`  | number | `10`          | Rotation angle along the Z-axis (applied statically when hovered).          |
| `children`| ReactNode | -         | The content to which the 3D effect will be applied.                         |
